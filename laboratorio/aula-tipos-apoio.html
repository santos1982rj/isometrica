🏗️ Vínculos e Reações nos Apoios

Fundamentos de Estática nas Estruturas — Isométrica

O que são vínculos?

Vínculos (ou apoios) são dispositivos que conectam uma estrutura ao mundo exterior, restringindo seus movimentos. Cada movimento que um vínculo impede é chamado de grau de liberdade (G.L.) restringido.

Uma estrutura no plano tem 3 graus de liberdade:

  • Translação horizontal (movimento no eixo X)
  • Translação vertical (movimento no eixo Y)
  • Rotação (giro em torno do eixo Z)

Cada vínculo impede alguns desses movimentos e, como consequência, surgem forças de reação na direção do movimento impedido.

📌 Princípio fundamental: Para cada movimento impedido, surge uma força de reação correspondente. Translação impedida → força (H ou V). Rotação impedida → momento (M).

Os três tipos de apoio

✗ Translação Y

Cada movimento impedido gera uma reação: translação → força, rotação → momento

Cálculo das reações

Para determinar as reações nos apoios, usamos as equações de equilíbrio da estática:

ΣFH = 0     ΣFV = 0     ΣM = 0

Essas três equações são suficientes para resolver estruturas isostáticas (aquelas cujo número de reações iguala o número de equações disponíveis).

Passo a passo para uma viga biapoiada

  1. Identificar os apoios — Na viga biapoiada, temos um apoio fixo (A) e um apoio móvel (B). O apoio fixo tem reações HA e VA. O apoio móvel tem apenas VB.
  2. Calcular as reações verticais — Usamos ΣM = 0 para eliminar uma das incógnitas. Por exemplo, ΣMA = 0 elimina HA e VA, deixando apenas VB.
  3. Calcular a reação horizontal — Se não há cargas horizontais, HA = 0.
  4. Verificar com ΣFV = 0 — A soma das reações verticais deve igualar a carga total aplicada.
📌 Dica: Use ΣM = 0 em um dos apoios para eliminar as incógnitas daquele apoio. Isso reduz o sistema para uma equação com uma incógnita.

Exemplo: Viga biapoiada com carga distribuída

Considere uma viga de vão L com uma carga uniformemente distribuída de intensidade q (kN/m). Vamos calcular as reações e traçar o diagrama de momento fletor.

Reações de apoio

Por simetria, cada apoio absorve metade da carga total:

VA = VB = q · L / 2     HA = 0

Diagrama de momento fletor (DMF)

Para uma carga distribuída, o momento fletor varia em parábola do 2° grau. O momento máximo ocorre no centro do vão:

Mmáx = q · L² / 8

Pela convenção da NBR 6118, o diagrama de momento positivo é desenhado do lado tracionado da viga. Como a carga é para baixo, as fibras inferiores são tracionadas — portanto, o diagrama é desenhado abaixo da linha de referência.

Exercícios propostos

  1. Uma viga biapoiada de vão L = 6 m recebe uma carga distribuída q = 12 kN/m. Calcule as reações verticais nos apoios A e B.

    VA = VB = q·L/2 = 12·6/2 = 36 kN

  2. Para a mesma viga do exercício anterior, calcule o momento fletor máximo.

    Mmáx = q·L²/8 = 12·6²/8 = 54 kN·m

  3. Uma viga engastada em uma extremidade (balanço) tem vão L = 3 m e carga distribuída q = 8 kN/m. Determine as reações no engaste.

    V = q·L = 8·3 = 24 kN (↑)
    H = 0 (sem cargas horizontais)
    M = q·L²/2 = 8·3²/2 = 36 kN·m (sentido anti-horário)
